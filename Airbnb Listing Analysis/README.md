# AirBnB Listing Analysis

This is a guided project I followed to practice my Pandas skills. It is part of the Python for Data Analytics course by Maven Analytics.

The goal is to summarize the factors that affect Airbnb prices and determine the impact of 2015 regulations on Airbnb listings in Paris.

The project is divided into three parts:

1. Explore the dataset and identify data quality issues.
2. Group data to prepare them for the visualization.
3. Visualize the findings.
