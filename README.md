
This repository is a collection of guided projects I completed to consolidate and demonstrate my technical skills in data analysis, SQL, and visualization. Each project contains data, analysis scripts/notebooks, and notes for reproducibility.

## **What you'll find**
- **Airbnb Listing Analysis**: Exploratory analysis and visualization of Airbnb listings and the impact of 2015 regulations. 
- **Baby Name Trends**: Historical and spatial analyses of baby name popularity using SQL and CSV data. 
- **Major Baseball League Insights**: SQL-based analysis of player careers, salaries, and educational backgrounds. 
- **Sales & Customer Analytics**: End-to-end SQL-focused analyses, notebooks, and helper scripts.
- **World Economic Report**: Notebook exploring world indicators (HDI and related data). 

## **Skills demonstrated**
- Data cleaning and preparation (CSV, SQL imports)
- Exploratory data analysis and visualization (Jupyter notebooks)
- SQL querying, view creation, and reporting
